# 桌面小狗机器人 (Dog Board)

## 概述

桌面小狗机器人是基于palqiqi代码开发的四足机器人，使用对角线蓄力方法实现行走。

## 硬件特点

- **4个舵机配置**：
  - 左后腿 (GPIO 39) - 原palqiqi左腿
  - 左前腿 (GPIO 38) - 原palqiqi左脚
  - 右前腿 (GPIO 17) - 原palqiqi右腿
  - 右后腿 (GPIO 18) - 原palqiqi右脚

- **舵机安装方式**：
  - 舵机转轴平行于地面
  - 脚可以垂直地面前后运动

## 运动原理

### 对角线蓄力方法

通过动作的**"快慢组合"**来打破摩擦力的平衡：

#### 前进动作

**第一步：对角线蓄力**
- 左前腿 + 右后腿：缓慢地向后方拨动（支撑腿，负责推动身体前进）
- 右前腿 + 左后腿：快速地向前方划动（摆动腿）
- **关键点**：由于移动速度快，摆动腿与地面的静摩擦力会被破坏，变成动摩擦力，阻力较小；而慢速拨动的腿抓地力更强

**第二步：重心交替**
- 当拨动到极限位置时，交换角色
- 原来向后拨的腿快速向前收，原来向前收的腿缓慢向后拨

#### 后退动作

运动方向相反，原理相同。

## 舵机映射

| Dog舵机 | GPIO | 原Palqiqi舵机 | 说明 |
|---------|------|---------------|------|
| 左后腿 | 39 | 左腿 | 后方左侧 |
| 左前腿 | 38 | 左脚 | 前方左侧 |
| 右前腿 | 17 | 右腿 | 前方右侧 |
| 右后腿 | 18 | 右脚 | 后方右侧 |

## MCP控制接口

- `self.dog.walk_forward` - 前进
- `self.dog.walk_backward` - 后退
- `self.dog.stop` - 停止
- `self.dog.home` - 回到休息姿态
- `self.dog.set_trim` - 设置舵机微调
- `self.dog.get_trims` - 获取舵机微调值
- `self.dog.get_status` - 获取状态

## 运动参数

- **steps**: 步数（默认4）
- **speed**: 每步周期（毫秒，默认1000）
- **amount**: 步幅大小（度数，默认30）
