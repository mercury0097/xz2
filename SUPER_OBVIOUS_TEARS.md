## 🔧 技术细节

### 颜色转换策略

```python
# 策略 1: 最亮区域 (brightness > 150)
→ 亮青色 #00FFFF (0, 255, 255) ★ 非常醒目！

# 策略 2: 中等亮度 (80 < brightness ≤ 150)
→ 浅天蓝 #87CEFA (135, 206, 250)

# 策略 3: 较暗区域 (30 < brightness ≤ 80)
→ 深青色渐变 (根据亮度调整)

# 策略 4: 很暗区域 (brightness ≤ 30)
→ 暗青色阴影 (微弱蓝光)
```

### 为什么选择 #00FFFF？
- **R=0, G=255, B=255**：纯青色，无红色分量
- **最高饱和度**：G 和 B 都是最大值 (255)
- **完美的"水"的颜色**：清澈、透明、流动
- **霓虹效果**：在深色背景上非常醒目
